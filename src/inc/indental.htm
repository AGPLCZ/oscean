The Indental {https://github.com/XXIIVV/Libraries/blob/master/scripts/indental.js parser} allows for human-readable data structures for static APIs. In the Indental file, an unindented line declares the key to a new root node, children lines can associate either parameters or lists to their parent node, a line divided with a colon will associate a value to a parameter to the parent node, and a sequence of equally indented lines will append to a list. You can download syntax highlight {https://github.com/hundredrabbits/Libraries/blob/master/tools/ndtl.sublime-syntax here}.

NAME
  KEY : VALUE
  LIST
    ITEM1
    ITEM2

Or, {NAME:{KEY:VALUE,LIST:[ITEM1,ITEM2])}
