The Tablatal {https://github.com/hundredrabbits/Libraries/blob/master/scripts/tablatal.js parser} allows for human-readable data structures for static APIs. In the Tablatal file, the first line declares the key, the spacing between each key defines the length of the parameters for all subsequent lines. You can download syntax highlight {https://github.com/hundredrabbits/Libraries/blob/master/tools/tbtl.sublime-syntax here}.

NAME    AGE   COLOR
Erica   12    Opal
Alex    23    Cyan
Nike    34    Red
Ruca    45    Grey

Or, [{name:Erica,Age:12,Color:Blue},{name:Alex,Age..}
