    Chr6 is a simple chr editor written in ANSI C.

    Chr6 is a simple version of Nasu designed to edit a single page of Famicom sprites. To control the window size and default colors, edit the values defined at the top of the file itself. To build chr6, you need to install SDL2, for a Plan9 compatible version, see chr9.

    sudo apt-get install libsdl2-devcc -std=c89 -Wall chr6.c -I/usr/local/include -L/usr/local/lib -lSDL2 -o chr6

    Press E to export to export.chr, and press h to toggle tile guides.
