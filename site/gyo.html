    Gyo is a tiny imaginary computer programmable with its own assembly language.

    Gyo was designed as an experiment to explore a system with a minimal instruction sets. The computer has 16 registers, 256 memory addresses, 16 opcodes, the assembler syntax supports ~constants, =variables and :labels. The entire system and assembler is written in less than 400 lines of ANSI C code.

    Instructions

    0x0BRKBreak
    0x1JMPJump to position
    0x2JEQJump to position, when carry
    0x3JNEJump to position, when not carry
    0x4RMEStore in register from addr in memory
    0x5RVAStore in register from value
    0x6MREStore in memory from register at value
    0x7MVAStore in memory from value at register
    0x8ADDAdd value to register
    0x9SUBSubstract value from register
    0xAEQUCompare, set carry when equal
    0xBLESCompare, set carry when less than
    0xCCLCClear carry
    0xDTELStore in register from register id
    0xESELSelect register
    0xFNOPDo nothing

    Example

    ; constants

~rate #01

; variables

=var1
=var2

; program 

:label
	SEL #00
	RVA #09
	SEL #01
	RVA #09

:example1           ; example conditional
	NOP
	SEL #00
	EQU #01         ; check if r0 is equal to r1
	JEQ there       ; if equal, goto There
:here
	MRE var1        ; write in addr$01
	JMP example2
:there
	MRE var2        ; write in addr$02
	JMP example2

:example2           ; example loop
	NOP
	RVA #02
	SEL #01         ; select r1
:loop
	EQU #00         ; set flag if r0 = r15
	SUB rate        ; subtract r0 by rate
	JNE loop        ; goto loop if not equal
	BRK
