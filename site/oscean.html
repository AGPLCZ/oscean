    Oscean is a flat-file wiki engine.

    Oscean is a static wiki engine written entirely inC99, designed to be deployed from low-power devices with gcc as its only dependecy. The engine has grown into a collection of tools, including the time tracking software horaire, as well as the time formats neralie & arvelie.

    The generated files use no javascript, are optimized for screen-readers and terminal browsers, the entire CSS content of the entire site should be under 1kb.

    The software architecture is inspired by longtermism.
